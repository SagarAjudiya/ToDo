
# ToDo


- Xcode - 15.3
- Swift - 5.9
- iOS - 17.2

How to install
-
- Create clone or download the zip file. Open the ToDo.xcodeproj file and run the code.

App Feature
-
- Create the app using an UIKit and stored the user's data into the UserDefaults.
- Add your task
- Edit your task
- Complete your task
- Delete your task
